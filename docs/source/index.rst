Быстрый старт
-------------

Веб-приложение для анализа криптовалютных данных с Flask-бэкендом.

Установка и запуск
------------------

.. code-block:: bash

   git clone <your-repo>
   cd crypto_app
   
   pip install -r requirements.txt
   python app.py
   
   http://localhost:5000
